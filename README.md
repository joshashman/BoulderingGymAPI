This is a RESTful web API developed using ASP.NET core used for managing a bouldering gym.
The API allows management of users, memberships, climbing routes, sessions, booking, along with user capability to log route attempts and like their favourite routes.

ASP.NET identity is used for authentication and authorisation, and Entity Framework Core with SQLite for database management.

Requirements for running this project:  
.NET 8 SDK  

Clone repository:  
git clone https://github.com/joshashman/BoulderingGymAPI  
cd BoulderingGymAPI

Run the API:  
dotnet run  

For testing API endpoints:  
http://localhost:xxxx/swagger
